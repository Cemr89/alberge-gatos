🐾 MiauAdopta - Albergue de Gatitos
MiauAdopta es una plataforma web diseñada para facilitar la adopción responsable de gatos rescatados. Este proyecto permite a los usuarios conocer a los felinos disponibles, informarse sobre el proceso de rescate y enviar solicitudes de adopción de manera directa al administrador.

🚀 Características
Galería de Gatitos: Visualización de los gatos disponibles para adopción como Simba, Luna y Félix.

Formulario Dinámico: Sistema de contacto para interesados en adoptar con validación de campos obligatorios.

Integración con EmailJS: Las solicitudes se envían automáticamente por correo electrónico sin necesidad de servidor (backend).

Interactividad: Incluye efectos de sonido personalizados (mew.wav) al interactuar con las tarjetas de los gatos.

Estado de Envío: Botón interactivo que muestra el estado "Enviando..." y lanza una alerta de confirmación al tener éxito.

🛠️ Tecnologías Utilizadas
HTML5: Estructura semántica del sitio y formulario.

CSS3: Estilos personalizados y diseño responsivo.

JavaScript (ES6+): Lógica para el manejo de eventos, sonidos y envío de datos.

EmailJS: Servicio externo para el manejo de correos electrónicos desde el navegador.
